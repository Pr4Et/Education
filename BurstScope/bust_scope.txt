Burst Scope for physical acoustic demonstations (geometric acoustics, time reversal mirror, deconvolution).
Uses a loudspeaker as emitter and a microphone as detector.
Designed by Shahar Seifer, Weizmann Insitute of Science, 2025-2026
license: CC-BY-4.0  (https://creativecommons.org/licenses/by/4.0/legalcode)

 - Stage 1: Estimates I/O latency using periodic bursts.
 - Stage 2: Synchronized burst scope, demonstrates reflections in the room.
 - Stage 3: Using controls:
     0 : Stop bursts, idle.
     1 : Generate impulse, record the response and store in memory.
     2 : Play memory in reverse, show microphone response.
     3 : Play deconvolution signal that should produce delta funcion at the microphone location, show response.
     r : Record 10s interval, searche for the hightest peak and store in memory (window centered at peak)
     q : Quit the program


Installation in Windows
Install Python (3.10+ recommended).
pip install sounddevice numpy matplotlib
python burst_scope.py
A dialog box will list your audio devices in a drop-down menu. Choose the preferred input and output devices (choose none for default).
Tip: For best results, use a USB audio interface with a proper mic input (mono is fine) rather than the PC’s built in mic jack. Any of the two loudspeaker channels can be used.
